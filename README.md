# WPF_LB1
Необходимо разработать приложение, содержащее набор кнопок, занимающих 2/3 ширины окна при любых его размерах. Каждая кнопка должна в качестве содержимого отображать название цвета и обладать наружным отступом равным 2.0. Также соответствующий цвет должен использоваться в качестве цвета переднего плана кнопки. <br>Необходимо использовать следующий набор цветов: Navy, Blue, Aqua, Teal, Olive, Green, Lime, Yellow, Orange, Red, Maroon, Fuchsia, Purple, Black, Silver, Gray, White.
